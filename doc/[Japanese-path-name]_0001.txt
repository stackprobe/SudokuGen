===============================================================================

　いろいろなナンプレの問題を作成して印刷するプログラム

===============================================================================


■ソフトの概要

　いろいろなナンプレの問題を作成して印刷するプログラムです。

　ナンプレの種類と難易度、印刷設定を入力して印刷ボタンを押せば、
　自動生成されたナンプレを簡単に印刷出来ます。

　プリンタが無くても動作しますが、あった方が良いです。


■動作環境

　Windows 7
　確認していませんが Windows 8/8.1/10 でも動くかもしれません。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　"ナンプレ作成印刷プログラム.exe" を実行して下さい。


■使い方

　●問題を作成して印刷する。

　　1. ナンプレの種類をリストから選択して下さい。

　　2. 問題の難易度をコンボボックスから選択して下さい。

　　   難易度は、"普通" < "普通＋" <<< "難しい" <<<<<<<<< "とても難しい" となっております。

　　3. "問題を作成して印刷" ボタンを押して下さい。

　　4. 使用するプリンタを選択する場合は、"デフォルトのプリンタを使用する" のチェックを外し、
　　   コンボボックスからプリンタを選択して下さい。

　　5. 用紙サイズを選択する場合は、"デフォルトの用紙サイズで印刷する" のチェックを外し、
　　   コンボボックスから用紙サイズを選択して下さい。

　　6. 余白を入力する場合は、"デフォルトの余白で印刷する" のチェックを外し、
　　   左上右下それぞれの余白を入力して下さい。

　　   単位は 1/100 インチです。例えば 150 と入力すると 1.5 インチになります。

　　7. 解答（問題の答え）も印刷する場合は、"解答を印刷する" にチェックを入れて下さい。

　　8. 印刷ボタンを押して下さい。

　　   問題の作成完了を待たずに印刷を開始出来ます。
　　   （この場合、問題の作成完了を待ってから印刷を開始します）


　●過去に作成した問題を印刷する。

　　1. ツール / 過去に作成した問題の一覧(R) を選択して下さい。

　　2. 印刷したい問題を選択して、印刷ボタンを押して下さい。

　　3. 後は、問題を作成して印刷したときと同じです。


■補足

　過去に作成した問題は 100 件まで保持します。

　合体ナンプレの重なった部分や特殊な升に数字が少ないと、より面白くなるかもしれません。
　ツール / 設定 / 詳細 / 合体ナンプレ〜 にチェックを入れると、
　合体ナンプレの重なった部分・対角線上にも数字が揃う問題の対角線上・偶数升・奇数升
　に、なるべく数字を置かないようにします。


■問題点というか既知のバグ

　難易度が安定せず、選択した難易度より簡単な問題が作成されてしまうことがあります。
　（選択した難易度より難しくなることはありません）
